  3. Visual C++ Build Tools或Visual Studio 2015
  4. .NET Framework 4.5.1（仅限Windows Vista/7）
  5. npm install -g node-gyp

BalmJS

Be inspired by Laravel Elixir

功能

  • 样式：sass/less/postcss
  • 脚本：es6/typescript
  • 雪碧图：支持
  • 缓存文件：支持
  • 代理服务器：支持
  • 支持的项目类型：html/laravel/vue.js/react/typescript+react/angular2等

安装

npm install --save-dev balm gulp

配置

// gulpfile.js
var balm = require('balm');

balm.config = {
  proxy: 'www.yiban.dev',
  roots: {
    source: 'src',
    target: 'public'
  },
  paths: {
    source: {
      css: 'css',
      js: 'js',
      img: 'img'
    }
  },
  styles: {
    ext: 'scss'
  },
  scripts: {
    entry: {
      'main': './app/js/main.js'
    }
  },
  sprites: {
    imgList: ['icon']
  }
};

balm.go();

使用

gulp // 开发
gulp --production // 打包
